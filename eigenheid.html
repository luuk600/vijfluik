Eigenheid

Bij Eigenheid heb ik mijn C++ project neer gezet omdat dit het meeste is welke richting ik op wil. Bij dit project was het idee om veel te leren over een programeer taal die ik niet kende. Eerst wou ik een van de moeilijkste programeertalen te kiezen maar ik kreeg dat nog net niet voor elkaar. Dit project deed ik in de 2de periode zo het was nog een beetje lastig. Ik heb ook veel geprobeerd in assembly. Met c++ heb ik wat opdrachten gemaakt die we ook in C# hebben gemaakt en het is me gelukt. Ik ben best ver gekomen. Het lastigste van dit project was de random getallen hier had ik wel even voor nodig.
